# 📖 Macro Tracking App - Development Steps

This document outlines the development process of the **Macro Tracking App** built with **React Native** and **Expo**. It serves as a reference for the project structure, implemented features, and future enhancements.

---

# 🚀 Project Setup

### 1. Clone Repository

```bash
git clone https://github.com/Naman762/Fullstack-macro-travking-app.git
cd Fullstack-macro-travking-app
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npx expo start
```

Run on:

* Android Emulator
* iOS Simulator
* Expo Go
* Web Browser
